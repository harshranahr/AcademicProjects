## 🚀 Featured Project: Online Voting System
**Level:** BCA Major Project (6th Semester)  
**Academic Institution:** Himachal Pradesh University

### 📝 Project Description
The Online Voting System is a secure web-based application designed to modernize the electoral process. It replaces traditional paper-based voting with a digital interface that ensures data integrity, voter accessibility, and real-time result processing.

### 🛠 Technical Stack
* **Frontend:** ASP.NET (C#)
* **Backend:** MySQL Database
* **Development Environment:** Visual Studio 2022
* **Documentation:** Microsoft Word (Technical Report)

### 👤 Individual Contribution & Role
As the **Project Lead**, I conceptualized the system architecture and managed the development lifecycle for a team of four. My specific technical contributions included:
* **Admin Module Development:** Engineered the administrative dashboard to handle candidate registrations and secure login credentials.
* **Database Architecture:** Designed the relational schema for candidate profiles, voter verification, and live ballot tracking.
* **Backend Logic:** Implemented C# server-side logic for automated result tallying and administrative moderation tools.
* **Validation Systems:** Integrated real-time input validation to verify voter eligibility and prevent duplicate entries.

### ✨ Key Features
* **Secure Authentication:** System-generated unique IDs for both voters and candidates.
* **Real-Time Analytics:** Instant result calculation and graphical data representation post-election.
* **Admin Oversight:** A dedicated moderator interface to approve or reject candidate documents before they appear on the ballot.
